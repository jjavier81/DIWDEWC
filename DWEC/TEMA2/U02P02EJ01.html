1. Crea un objeto “fecha” empleando el objeto Date. A partir de ese objeto:

var fechaActual = new Date();

  1. Muestra por consola el día (del 1 al 31)

    2. console.info("Día del mes: " + fechaActual.getDate());

  2. Muestra por consola el mes (del 1 al 12)

    3. console.info("Mes del año: " + (fechaActual.getMonth()+1));

  3. Muestra por consola el año

    4. console.info("Año actual: " + fechaActual.getFullYear());

  4. Muestra el día de la semana (Lunes, Martes, …)

    5. var diaSemana = new Array("Domingo", "Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado");
    console.info("Día de la semana: " + diaSemana[fechaActual.getDay()]);

  5. Muestra la fecha completa con formato dd/mm/aaaa

    6. console.info(fechaActual.getDate() + "/" + (fechaActual.getMonth()+1) + "/" + fechaActual.getFullYear());

  6. Muestra las horas (de 0 a 23)

    7. console.info("Hora actual: " + fechaActual.getHours());

  7. Muestra la fecha completa con formato dd/mm/aaaa HH:MM:SS

    8. console.info(fechaActual.getDate() + "/" + (fechaActual.getMonth()+1) + "/" + fechaActual.getFullYear() + " " + fechaActual.getHours() + ":" + fechaActual.getMinutes() + ":" + fechaActual.getSeconds());
